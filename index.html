HOW IT WORKS

The 4-Phase Cycle

Transform prompts through a systematic lifecycle

PHASE 01

Creation

Build a crystal-clear base prompt with structured XML tags. Define objective, context, requirements, steps, output format, and verification criteria.

PHASE 02

Optimization

Elevate with frontier tactics. Flaw audit, inject amplifiers (CoT, ToT, personas), clarify output requirements, add self-critique loops.

PHASE 03

Deployment

Model-tuned execution plan. Select optimal LLM, choose execution mode, configure parameters, execute and capture output.

PHASE 04

Iteration

Track progress and improve. Score output quality, verify against checklist, log cycle, refine based on learnings.

Get Started in 3 Minutes

Simple setup, powerful results

Step 1: Get the Framework

Download prometheus-cycle-v2.md from the GitHub repository. It's a single file containing the complete framework.

Step 2: Paste as System Prompt

Copy the entire contents and paste it as your system prompt or first message in your LLM (ChatGPT, Claude, Grok, Gemini).

Step 3: Describe Your Task

Simply describe what you want to accomplish. PROMETHEUS will guide you through all 4 phases automatically.
