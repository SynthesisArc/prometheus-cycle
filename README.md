## üöÄ Quick Start (3 Minutes)

### Step 1: Copy the Framework

Copy the entire contents of [`prometheus-cycle-v4.0.md`](prometheus-cycle-v4.0.md)

### Step 2: Paste as System Prompt

In your LLM of choice (ChatGPT, Claude, Grok, etc.), paste PROMETHEUS as your **system prompt** or **first message**.

### Step 3: Send Your Task

Simply describe what you want to accomplish:

```
"Write a professional email declining a meeting request"
```

or

```
"Create a LinkedIn post about AI adoption in healthcare"
```

or

```
"Build a prompt for an AI agent that triages GitHub issues"
```

### Step 4: Watch the Magic

PROMETHEUS will guide you through all 5 phases automatically, producing a production-ready prompt with:
- Discovery confirmation (`<task_type>`, `<success_looks_like>`, `<stakes>`)
- Clear structure (`<persona>`, `<objective>`, `<context>`, `<examples>`, `<requirements>`, `<reasoning_strategy>`, `<steps>`, `<output>`, `<verification>`)
- Track-specific sections (Content: `<voice>`, `<audience>` | Code: `<technical_context>`, `<edge_cases>` | etc.)
- Advanced reasoning strategies applied
- Model-specific optimization
- 9-criterion quality scoring

**That's it!** You now have a professional-grade prompt ready for production.

3. **CODE TRACK** - Development, debugging, architecture (`<technical_context>`, `<edge_cases>`, `<tests>`, `<maintainability>`)

5. **MULTIMODAL TRACK** - Images, video, audio, mixed (`<modality_order>`, `<visual_context>`, `<grounding>`, `<temporal>`)

### Reasoning Strategies

Choose the right thinking approach for your task:

- **Chain-of-Thought (CoT)** - Linear step-by-step reasoning (35% fewer errors)
- **Tree-of-Thoughts (ToT)** - Explore multiple approaches, select best
- **Graph-of-Thoughts (GoT)** - Synthesize disparate insights (62% quality boost)
- **Skeleton-of-Thought (SoT)** - Outline first, expand in parallel (2x faster)
- **Algorithm-of-Thoughts (AoT)** - Algorithmic search strategies (10x fewer queries)
- **Buffer-of-Thoughts (BoT)** - Reusable reasoning patterns
- **Chain-of-Verification (CoVe)** - Self-correction (40% fewer hallucinations)
- **Self-Consistency** - Consensus-based confidence
- **Reflexion** - Learning from failure

### Model Adapters

Concrete optimization guidance for:

- **Claude** - XML-native, prefilling, `<thinking>` tags
- **GPT-4** - Literal following, sandwich method, structured outputs
- **Gemini** - Context first, anchor phrases, caching
- **Grok** - Truth-seeking, direct, less hedging
- **Open-Source** - Chat template requirements
